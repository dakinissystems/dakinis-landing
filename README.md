# Dakinis Systems — Landing

Página corporativa (enterprise) en React + Vite + Tailwind para Dakinis Systems.

## Repositorio

[https://github.com/ChristianDVillar/DakinisSystems](https://github.com/ChristianDVillar/DakinisSystems)

## Desarrollo local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Despliegue

Sube la carpeta `dist/` a tu hosting estático (Vercel, Netlify, GitHub Pages, etc.).
